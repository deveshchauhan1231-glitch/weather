# Weather Website

A simple weather website that displays real-time weather information using data from **Open-Meteo**.

## Description
This project fetches weather data from the Open-Meteo API and presents it in a clean, user-friendly interface. It is built using basic web technologies and focuses on simplicity and responsiveness.

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Open-Meteo API  

## Features
- Fetches live weather data  
- Displays temperature and weather conditions  
- Responsive design  
- Lightweight and fast (no external frameworks)

## How It Works
1. User enters or selects a location  
2. JavaScript fetches weather data from Open-Meteo  
3. Data is processed and displayed dynamically on the webpage  

## Setup
1. Clone the repository  
2. Open `index.html` in your browser  
3. No additional setup required  

## API
Weather data is provided by the **Open-Meteo API**.
